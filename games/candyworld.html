Candy World

Development
The game is developed originally in 2015 using javascript and unity.
Unity later removed javascript support which meant all code had to be rewritten in c#.
Unfortunatly the game is incomplete because of this.
All original models, animations and texture done by me using Maya and Photoshop.

Controls and Objectives
WASD to move around and E to attack.
Collect keys to open locked doors.
Find the baker town and kill the boss.
